HTML & CSS Quick-Reference Guide

HTML Tags

<h1>- Header-1 Tag

Description: h1 Tag defines the most important heading in the document. It is typically used for the main title or topic by providing a clear structure and helping users understand the focus of the content on the page.

<h1>Welcome To My Website</h1>

<p>- Paragraph Tag

Description: The p tag is used to define a paragraph of text. It automatically adds space before and after the text, helping to organize content into readable sections and improving the overall layout of a webpage.

<p>My website is your go to place for all the latest updates and info. Explore our content and enjoy your stay!</p>

<a>-Anchor(link) Tag

Description: tag is used to define hyperlinks, allowing users to navigate from one page to another or to a specific part of the same page. It can link to external websites, other pages within the same website or even specific resources like email addresses or files

<a href="www.saharaswebsite.com>Click here</a>

<img>- Image Tag

Description: img tag is used to embed images in a webpage. It requires the "src" attribute to specify the image source and the "alt" attribute to provide alternative text for accessibility. Images help convey information effectively.

<img src="myimage.jpg" alt="Scenary">

<div>- Division Tag

Description: Div tag is a block-level element used to group content and structure a webpage. It helps organize elements for styling and layout purposes.


 <div> 
 <h2>Portfolio</h2> 
 <p>Hello I am Sahara!</p> 
</div>

<ul> and <li>- Unordered List and List Item Tags

Description: Unorder List tag is used to create an unordered list where items are displayed with bullet points.
List Item Tag defines an individual item within a list, whether itâ€™s an unordered or ordered list. It helps organize content into structured lists.

<ol> and <li>- Ordered List and List Item Tags

Description: The Ordered tag is used to create an ordered (numbered) list, where each item is automatically numbered. It is useful when the order of the items is important such as steps in a process or ranking items.

<ol>
 <li>First</li> 
 <li>Second</li> 
 <li>Third</li> 
</ol>

<Strong>- Strong Tag

Description: The Strong tag is used to define text with strong emphasis which is typically displayed as bold text. It is generally used for important content.

<p>This message is very<Strong>Important</Strong>!!</p>

<footer>- Footer Tag

Description: The Footer tag is used to define the footer section of a webpage or a section of content. It typically contains information like the copyright notice, contact information, privacy policy links or other relevant details that usually appear at the bottom of a page.

<footer>© By Sahara</footer>

<hr>- Line Break

Description: The br tag is used to insert a line break within text, making content appear on a new line. It does not have a closing tag.

<p>This is the first line. <br> This is the second line.</p>

<hr>- Horizontal Rule

Description: The hr tag is used to create a horizontal line to separate sections of content. It helps to visually divide content on a page.

<h2>Section 1</h2>
 <p>This is the first section.</p> 
 <hr> 
 <h2>Section 2</h2> 
 <p>This is the second section.</p>

CSS Properties

Color

Description: The color property is used to set the text color of an element.


        p{
          color: white;
        }

Background Color

Description: The background color property is used to set the background color of an element. It can apply to the entire webpage or specific elements


        p{
            background-color: pink;
        }

Font Size

Description: The font size property is used to define the size of text in an element using differenr units.


        p{
            font-size: 20px;
        }

Font family

Description: The font family property is used to specify the font of the text. You can list multiple fonts as a fallback system in case the preferred font is not available on the user's device.


        p{
            font-family: 'Times New Roman', Times, serif;
        }

Text Align

Description: Aligns text horizontally within an element. Values include left, right, center and justify.


        p{
            text-align: center;
        }

Width

Description: Specifies the width of an element.


        p{
            width: 60%;
        }

Height

Description: Defines the height of an element.


        p{
            height: 300px;
        }

Border

Description: Sets the style, width and color of an element's border. It can be defined with values like solid, dashed, dotted along with thickness and color.


        p{
            border: 5px solid black;
        }

Margin

Description: Adds space outside an element by creating distance between it and other elements. It can be defined for all sides (top, right, bottom, left).


        div{
            margin: 15px;
        }

Padding

Description: Adds space inside an element, i.e, between its content and border. It can be set for all sides (top, right, bottom, left).


        div{
            padding: 20px;
        }
